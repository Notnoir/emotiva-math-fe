# 3D Models for AR Viewer

This directory contains GLB/GLTF 3D models for Augmented Reality visualization.

## Required Models:

- `cube.glb` - Kubus (Cube)
- `box.glb` - Balok (Rectangular Prism)
- `sphere.glb` - Bola (Sphere)
- `cylinder.glb` - Tabung (Cylinder)
- `cone.glb` - Kerucut (Cone)
- `pyramid.glb` - Limas (Pyramid)
- `prism.glb` - Prisma (Prism)

## How to Generate Models:

You can use tools like:

1. **Blender** - Free 3D modeling software, export as GLB
2. **Three.js Editor** - Create and export geometries
3. **Online GLB Generators** - Various web tools available
4. **Programmatically** - Use Three.js GLTFExporter to convert geometries

## Model Requirements:

- Format: GLB (binary glTF)
- Size: < 1MB per model
- Optimized for mobile AR
- Centered at origin (0,0,0)
- Unit scale: 1 unit = 1 meter

## Quick Start:

For testing purposes, you can use simple placeholder GLB files or generate them from basic Three.js geometries.
